# AI On-Chain Anomaly Detector

## Overview
This project detects anomalous wallet behavior using AI-inspired heuristics.

## Features
- Transaction pattern analysis
- Risk scoring (Normal, Suspicious, High Risk)
- Lightweight and extendable

## Why It Matters
With the rise of AI-driven blockchain systems, detecting anomalous behavior is critical for preventing exploits and financial losses.

## Future Improvements
- Integration with real blockchain APIs
- Machine learning models (Isolation Forest)
- Real-time monitoring dashboard
